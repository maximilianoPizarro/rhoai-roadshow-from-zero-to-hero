## Welcome to the Journey

This workshop takes you through a complete journey from understanding the business use case to deploying a production-ready AI solution on Red Hat OpenShift AI.

![Workshop Overview](../images/portada-team.png)

## What You'll Experience

Throughout this workshop, you'll:

- **Understand the Business Case**: Learn about Neuralbank's challenges and how AI can solve them
- **Set Up Your Environment**: Connect to OpenShift AI and configure your development workspace
- **Build MCP Agents**: Create intelligent agents that interact with backend services
- **Deploy and Integrate**: Connect your agents to the frontend and test end-to-end
- **Monitor and Observe**: Use OpenTelemetry to track distributed traces

## The Complete Picture

The workshop covers the full stack:

- **Frontend**: Customer and commercial agent interfaces
- **Backend Services**: Java-based microservices
- **MCP Agents**: Intelligent agents using Model Context Protocol
- **Authentication**: Keycloak for identity management
- **Service Mesh**: Connectivity Link for secure communication
- **Observability**: OpenTelemetry for distributed tracing
