# MCP Agents for Neuralbank

Welcome to the MCP Agents section! Here you'll learn how to build, deploy, and integrate MCP (Model Context Protocol) Agents for Neuralbank's credit risk management system.

## What You'll Learn

In this section, you will:

- Understand MCP Agents and their role in Neuralbank
- Use the Golden Path and Developer Hub to bootstrap your project
- Set up a DevSpaces workspace for development
- Build an MCP Agent that queries and updates credit risk
- Test your agent with MCP Inspector and Cursor
- Deploy and integrate with the frontend
- Monitor operations with OpenTelemetry

## The Journey

As Kevin, you'll follow the Platform Engineering team's **Golden Path** - a curated set of best practices and standards that make it easy to:

- Generate development environments
- Access pre-configured Software Templates
- Develop directly on OpenShift using DevSpaces
- Deploy with confidence
